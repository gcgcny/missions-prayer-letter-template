# Assets Directory

This directory contains images and graphics for your missions fundraiser page.

## Required Assets

Replace the placeholder files with your own custom assets:

### 1. **map-placeholder.svg**
- **Purpose**: Map showing your mission destination
- **Recommended format**: SVG (scalable) or PNG
- **Suggested size**: 400x400px or larger
- **Tips**:
  - Use online tools like [MapChart](https://www.mapchart.net/) to create custom maps
  - Highlight your destination region in your theme colors
  - Keep it simple and clear

### 2. **dates-placeholder.svg**
- **Purpose**: Display departure and return dates
- **Recommended format**: SVG or PNG
- **Suggested size**: 400x200px
- **Tips**:
  - Use your theme colors
  - Make dates large and readable
  - Can include flight icons or calendar graphics

### 3. **statistics-placeholder.svg**
- **Purpose**: Visual representation of mission statistics (e.g., pie charts showing Christian population percentage)
- **Recommended format**: SVG or PNG
- **Suggested size**: 300x300px
- **Tips**:
  - Use tools like [Canva](https://www.canva.com/) or [Chart.js](https://www.chartjs.org/) to create charts
  - Match your theme colors
  - Keep data simple and impactful

### 4. **team-photo-placeholder.svg**
- **Purpose**: Photo of your missions team
- **Recommended format**: JPG or PNG
- **Suggested size**: 1200x800px (landscape orientation)
- **Tips**:
  - Use a high-quality group photo
  - Ensure good lighting and everyone is visible
  - Compress the image to reduce file size (use tools like [TinyPNG](https://tinypng.com/))

## Optional Assets

You can add additional images as needed:
- Individual team member photos
- Photos from previous mission trips
- Graphics for prayer requests
- Custom icons or illustrations

## Image Optimization Tips

1. **Compress images** to reduce load times (aim for < 500KB per image)
2. **Use SVG** when possible for logos and icons (infinitely scalable)
3. **Use descriptive file names** (e.g., `team-photo-2024.jpg` instead of `IMG_1234.jpg`)
4. **Test on mobile** to ensure images look good on small screens

## Tools & Resources

- **Map Creation**: [MapChart](https://www.mapchart.net/), [Google My Maps](https://www.google.com/maps/d/)
- **Graphic Design**: [Canva](https://www.canva.com/), [Figma](https://www.figma.com/)
- **Image Compression**: [TinyPNG](https://tinypng.com/), [Squoosh](https://squoosh.app/)
- **Icon Libraries**: Font Awesome (already included), [Noun Project](https://thenounproject.com/)
